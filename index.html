Namespaces


The root namespace for the packages are https://schema.gtsc.io/package-name/.


e.g. https://schema.gtsc.io/framework/


The JSON schema definitions for each type can be found at https://schema.gtsc.io/package-name/type.


e.g. https://schema.gtsc.io/aig/AuditableItemGraphVertex


Performing requests to the root namespace with the request Accept header set to application/ld+json will return the JSON-LD version of the types.


This performs a redirect to https://schema.gtsc.io/package-name/types.jsonld


e.g. https://schema.gtsc.io/aig/types.jsonld
